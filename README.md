# ☠️ Некромант-Феодал

Roguelike карточная веб-игра. Вы — некромант, только что вступивший во владение небольшим замком нежити. Советник Костян вводит вас в курс дела. Пройдите 25 ситуаций, не потеряв золото и войско нежити — и станете монархом процветающего королевства.

## Запуск

Открыть `index.html` в браузере. Никаких зависимостей, никакого сервера.

## Ресурсы

- 💰 **Золото** — экономика феодала
- 💀 **Нежить** — ваша армия и рабочая сила  
- 👁️ **Репутация** — скрытый параметр, влияет на доступные события

## Стек

Чистый HTML/CSS/JS, Google Fonts. Один файл `index.html`.
